Solving Guides!

Looking for solving guides for Ciphers? Here are three good options:

  1. We have many excellent guides written by ACA members that should put you on the road to solving ciphers faster.
  2. We have a full guide to all the ciphers. Updated 22-Jan-2024.
  3. Also don't forget the resources listed on the National Science Olympiad CodeBusters site
  4. We have a new YouTube Channel @Codebusters-sciol where we are posting short videos showing how to use the different cipher tools

Cipher usage for Division B/C this year.

If you are trying to figure out what ciphers will be used for what type of event, we have a nice Summary of Ciphers used for the 2023-2024 Season that was made by reading the current rules. As always, any official clarifications or rule updates override anything on that page, but it is a good start.

Summary of what's new:

  • Version 9.0.0 -- MAJOR UPDATE!
    • Combined with the Test Builder, all of the cipher types now provide suggested question text and score values based on the analyzed difficulty of the question.
    • Please note: if you find that the recommended scores are too high/low for a given phrase, please enter an issue at github.com/toebes/ciphers/issues. In order to not put the quote in public, you can mail it to codebusters@toebes.com referencing the issue number.
    • Ciphers now include a Quote Author field for the quote so that the recommended question text can include the source of the quote
    • The Baconian words cipher has buttons to automatically set the groupings of A/B values
    • The Baconian sequence cipher can suggest unicode combining overstrikes for the A/B text
    • Aristocrats/Patristocrats/Xenocrypts have support for suggesting K1/K2/K3 keywords
    • Aristocrats have a new dialog for finding homonyms in phrases and introducing typographical errors when creating a Misspelled cipher.
    • The Cryptorithm generator has been sped up multiple orders of magnitude when checking addition/subtraction problems.
      • The Test Builder phrase suggestion now allows for keywords to filter the quote search.
  • Version 8.10.0
    • We added a new tiny answerkey option to allow printing all of the answers in only a couple of pages to make it easier when grading. The new Test Builder propagates special bonus and Xenocrypt translations.
  • Version 8.9.0
    • The new Test Builder is available along with the Quote Manager. Together you can upload quotes from a spreadsheet and generate a test and automatically suggest appropriate quotes to use on the test.
    Previous Versions...

An quick tutorial of using the tool can be seen at https://www.youtube.com/watch?v=pcz_3ql8ebM

Please report any problems at github.com/toebes/ciphers/issues

Are you an Event leader looking to create a Division A/B/C test? You can use the new Test Builder as a starting point for a good mix of questions. Just follow the steps to create a test with questions and a guide to picking plain text for the ciphers.

Are you an Coach looking to create a Division A/B/C practice test? You can use the new Test Builder and the Quote Manager loading up the Previously Used English Quotes and Previously Used Spanish Quotes as a starting point to build your own sample tests. A video walking you through the process can be seen at https://youtu.be/3Nbv5Qsq51s
