Cipher usage for Division B/C this year.

If you are trying to figure out what ciphers will be used for what type of event, we have a nice Summary of Ciphers used for the 2022-2023 Season that was made by reading the current rules. As always, any official clarifications or rule updates override anything on that page, but it is a good start.

If you are looking for a resource sheet to print out for practice, you can find it here.

Interactive Tests are offline.

Now that we the event is back to paper, we are no longer offering the online test version. It isn't just a simple aspect of code, we could no longer justify the cost of the AWS server behind it.

New Ciphers!

We have added the Dancing Men cipher for Division A and it should be fully functional

For Division B/C we have added a preliminary version of the Fractionated Morse and Cryptarithm ciphers. Both allow for you to enter a cipher, but neither will currently generate the automated solution.

Are you a coach looking for hints and solving guides? Here are three places to go:
  • Use the Answers and Solutions automatically generated for any question
  • Use the resources listed on the national Science Olympiad CodeBusters site
  • The @NCSO_cb twitter feed includes guides for solving just about every type of problem in the event
Are you an Event leader looking to create a Division C Regional or Invitational test? You can use this sample test as a starting point for a good mix of questions and nominal score values. Don't forget to Randomize the Aristocrats and pick new values for the questions. You can also randomize the order of the questions when editing the test.
